E-Commerce Sales Performance Analysis (SQL & Python Portfolio Project)

This project is a comprehensive sales performance and profitability audit using an SQLite transactional database (database.db) containing e-commerce superstore records. It runs strategic business queries using SQL and translates them into actionable visual intelligence using Python in a Jupyter Notebook.



📌  Project Overview

As a Data Analyst, the goal of this project is to analyze raw historical transactions to identify:

Core KPIs: Revenue performance, net profit margin, and transaction density.

Product Performance: Top-performing categories and high-margin sub-categories.

Customer Value: High-impact customer segments driving corporate revenue.

Regional Trends: High-performing geographic regions vs. regions requiring operational support.

Temporal Seasonality: Monthly performance trends over the years to assist inventory management.


🛠️ Technical Stack & Frameworks

Database Management: SQL 

Data Processing & Connection: Python, sqlite3, pandas

Data Visualization: matplotlib, seaborn

Development Environment: VS Code & Jupyter Notebooks
